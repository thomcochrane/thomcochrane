Other Web 2 Media Sharing

Collaborative and reflective e-learning tools (e.g. blogs, wikis, RSS, instant messaging, podcasting, social book marking, etc...). These are often called "social software" or web 2 tools. The project links the use of freely available mobile friendly web 2 tools accessed via a smartphone with the learning objectives.
