YouTube: Video blogging



YouTube (http://www.youtube.com ) is currently the most popular video sharing site. The mobile version (http://m.youtube.com) supports viewing of videos online in the mobiles web browser, or via a downloadable Java client for specific phones. Uploading mobile videos to YouTube is achieved via email attachments or as a Shozu destination.

Pedagogy:

Event, data and resource capturing and collaboration. Creativity.
